# 📘 About This Repository

This repository documents my learning journey for the **Microsoft SC-900: Security, Compliance, and Identity Fundamentals** certification.

The goal of this project is to build a strong foundation in:
- Cybersecurity principles
- Microsoft security solutions
- Identity & access management
- Compliance concepts
- Cloud security fundamentals
- Zero Trust architecture
- Threat protection

This repo also serves as a long-term cybersecurity knowledge base and portfolio project.

---

# 🎯 Objectives

- Prepare for the SC-900 certification exam
- Develop foundational cybersecurity knowledge
- Learn Microsoft security and compliance tools
- Practice documentation and version control
- Build a structured study system using Obsidian + GitHub
