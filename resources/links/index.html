    Microsoft Learn — Free Learning Paths

    These are the official free learning paths that map directly to the SC-900 exam:

    Learning Path Topics Covered
    Security, Compliance, and Identity Fundamentals Zero Trust, defense-in-depth, encryption, identity concepts
    Microsoft Entra Capabilities Entra ID, MFA, Conditional Access, PIM, Identity Protection
    Microsoft Security Solutions Defender XDR, Sentinel, Azure security services
    Microsoft Compliance Solutions Purview, DLP, Compliance Manager, eDiscovery

    Tip: Complete all four learning paths on Microsoft Learn before taking the exam. They are free, interactive, and directly aligned to what Microsoft tests.
